# MELD-Multimodal-Emotion-Analysis
本專案為多模態情緒分析系統，使用 MELD 資料集，應用 PyTorch 進行情緒分類。

# 專案目標
建立一個能處理語音、文字與視覺資訊的多模態模型，準確辨識對話中的情緒類別。

- MELD Dataset
